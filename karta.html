Karta is a Pythonic toolkit for small- and medium-sized geospatial data tasks

Vector data

Karta computes properties and relationships between vector geometries such as points, lines, and polygons. Performance relies on Cython-accelerated methods.

Raster data

Karta simplifies management of in-memory digital elevation models, satellite imagery, and land classification maps and enables simple sampling and transformation operations.

Geodesy

Karta is coordinate-system aware and performs forward and inverse geodetic calculations either internally or by leveraging pyproj.

API Docs PyPI Github

Vector geometries

Bounding boxes, distances and lengths, intersection calculations, convex hulls

karta.Point, karta.Multipoint, karta.Line and karta.Polygon are all subclasses of karta.Geometry. All geometry objects can contain singlepart and multipart metadata, as well as a reference to a coordinate reference system object. Methods of geometry objects handle basic queries about geometry properties and relationships.

GeoJSON I/O is handled natively while ESRI shapefile access is thanks to pyshp. Geometries implement the __geo_interface__ attribute, permitting interchange with other libraries such as Shapely.

point = Point((-130.0, 52.0), crs=LonLatWGS84)

line = read_geojson("linedata.json")

polygon = Polygon([(-515005.78, -1301130.53),
                   (-579174.89, -1282271.94),
                   (-542977.83, -1221147.82),
                   (-437864.05, -1251641.55),
                   (-438160.72, -1252421.48),
                   (-437961.28, -1285314.00)],
                   crs=NSIDCNorth)

polygon.to_shapefile("outline.shp")

Raster grids

Grid statistics, resizing, resampling, and profile extraction

Regularly-spaced grids are represented by karta.RegularGrid, with data backed by numpy arrays. Grids with arbitrary spacing can be represented by the somewhat-experimental karta.WarpedGrid class. Grids can be clipped, resampled, queried at geographical points, and read or written to ESRI ASCII files or to GeoTiff via GDAL.

grid = read_gtiff("LC80360272015200.tif")
grid = grid.resample(100, 100)
plt.imshow(grid.values, extent=grid.get_extent(), origin="bottom")

Coordinate reference systems and geodesy

Placing data on Earth

Geometries and Grids are referenced to coordinate reference system objects (karta.crs.CRS) that provide information about the geographical projection and geodetic relationships between positions. Distances and areas on planes, spheres, and ellipsoids are handled in Karta, while coordinate projection relies on pyproj via the Proj4CRS subclass.
