Ocean data analysis

Data are organized into classes from narwhal.cast (profiles), narwhal.bathymetry (depths) and narwhal.geo (locations). Class methods and additional functions in narwhal.analysis exist for:

  • interpolation
  • baroclinic modes
  • water type inversion
  • buoyancy frequency
  • ellipsoidal geodesy

GSW integration

Narwhal provides a ctypes wrapper for the Gibbs Seawater Toolbox in the narwhal.gsw submodule. It can be used with or without the rest of Narwhal, as in:

density = narwhal.gsw.rho(cast["sa"], cast["ct"], cast["p"])

Visualize results

The narwhal.plotting submodule contains functions for creating property diagrams, cast profiles, and section plots.

Examples:

  • T-S plot
  • Section plot

Storage and data sharing

Data should not be tied to software. Therefore, data are serialized to JSON or HDF files. These common formats are open and easily imported into other analysis packages (such as MATLAB), or browser-based visualization libraries (such as D3).

Parsers are included for WOCE CTD casts in NetCDF format and Arctic Switchyard CTD casts in ASCII table format.
