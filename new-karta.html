Karta is for...

Vector Data

Vector data includes geometries such as points, lines, and polygons. Karta computes properties and relationships between geographical vector geometries.

Raster Data

Digital elevation models, satellite imagery, and land classification maps are all examples of raster data. When these fit within memory, Karta handles these with karta.Grid classes backed by Numpy arrays.

Geodesy

As an important part of processing geographical data, Karta is coordinate-system aware and performs forward and inverse geodetic calculations, either internally or by leveraging pyproj.

Karta is a Pythonic toolkit for small-to-medium sized geospatial data tasks

Vector geometries

Bounding boxes, distances and lengths, intersection calculations, convex hulls.

karta.Point, karta.Multipoint, karta.Line and karta.Polygon are all subclasses of karta.Geometry. All geometry objects can contain singlepart and multipart metadata, as well as a reference to a coordinate reference system object. Methods of geometry objects handle basic queries about geometry properties and relationships.

GeoJSON I/O is handled natively, while bindings to pyshp give access to ESRI shapefiles. Geometries implement the __geo_interface__ attribute, permitting interchange with other libraries such as Shapely.

point = Point((-130.0, 52.0), crs=LonLatWGS84)

line = read_geojson("linedata.json")

polygon = Polygon([(-515005.78, -1301130.53),
                   (-579174.89, -1282271.94),
                   (-542977.83, -1221147.82),
                   (-437864.05, -1251641.55),
                   (-438160.72, -1252421.48),
                   (-437961.28, -1285314.00)],
                   crs=NSIDCNorth)

polygon.to_shapefile("outline.shp")

Raster grid

Grid statistics, resizing, resampling, and profile extraction

Regularly-spaced grids are represented by karta.RegularGrid. Grids with arbitrary spacing can be represented by the somewhat-experimental karta.WarpedGrid class. Grids can be clipped, resampled, queried at geographical points, and read or written to ESRI ASCII files or to GeoTiff via GDAL.

grid = read_gtiff("glacier_landsat.tif")
grid = grid.resample(100, 100)
plt.imshow(grid.values, extent=grid.get_extent(), origin="bottom")
